change_calculator.c
Inaya Siddiqui

This program prompts the user to enter their purchase total and the amount paid,
and returns their change, or the difference between the purchase value and the amount paid.

To compile the program, run the command:
gcc -o change_calculator change_calculator.c

to run the program use the command:
./change_calculator
and enter your purchase total and the amount paid.
